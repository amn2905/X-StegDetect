# X-StegDetect: Explainable AI Steganography Forensic Portal

X-StegDetect is a digital forensic analysis web application designed to detect steganographic payloads generated by the state-of-the-art **ViT-DiffSteg** (Vision Transformer and Diffusion model) framework and explain model predictions using Explainable AI (XAI) overlays.

Developed for cybersecurity researchers, digital forensic investigators, and academic demonstrations, the portal combines deep learning classifiers with spatial residual analysis filters to isolate embedding patterns.

---

## 🌟 Key Features

1. **Robust Detection Engine**: Detects whether an image is a clean `COVER` or has a hidden payload (`STEGO`) with confidence scores.
2. **Deep Ensemble Analysis**: Supports Single Model checks (ResNet50, EfficientNet, ViT, Swin) or Ensemble Mode leveraging **Majority**, **Soft**, and **Weighted** voting strategies.
3. **Forensic Residual Filters**: Deconstructs image layers to expose:
   * **Noise Residual**: Isolate high-frequency noise by Gaussian-blur subtraction.
   * **Edge Residual**: Highlight boundary textures using Sobel filters.
   * **Artifact Map**: Map local standard deviation spatial entropy.
4. **Three Pillars of Explainable AI (XAI)**:
   * **Grad-CAM**: Highlight model attention zones (CNN and Transformer tokens).
   * **SHAP**: Game-theoretic Shapley values calculated over superpixel coalitions.
   * **LIME**: Local surrogate pixel regression to identify supportive and contradictory evidence.
5. **PDF Report Generator**: Compiles case details, original images, residual maps, and XAI charts into a printable forensic document.
6. **Docker Orchestration**: Multi-container support for automated, cross-platform deployment.

---

## 🛠️ Technology Stack

* **Backend**: Python, FastAPI, PyTorch, OpenCV, NumPy, Pillow, ReportLab
* **Explainability (XAI)**: SHAP, LIME, Grad-CAM (Hook-based)
* **Frontend**: React, Vite, Tailwind CSS, Chart.js, Axios
* **Deployment**: Docker, Docker Compose

## 🚀 Local Installation & Setup

### Prerequisites

* Python 3.10+
* Node.js 18+ (with npm)
* *Optional*: Docker and Docker Compose

---

### Step 1: Run Backend API

1. Navigate to the `backend/` directory:
   ```bash
   cd backend
   ```
2. Create and activate a virtual environment:
   ```bash
   python -m venv venv
   # Windows
   .\venv\Scripts\activate
   # macOS/Linux
   source venv/bin/activate
   ```
3. Install python requirements:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the models downloader to fetch standard weights:
   ```bash
   python init_models.py
   ```
5. Start the FastAPI development server:
   ```bash
   uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
   ```
   The backend will be running at `http://127.0.0.1:8000`. You can inspect interactive API docs at `http://127.0.0.1:8000/docs`.

---

### Step 2: Run Frontend App

1. Navigate to the `frontend/` directory in a new terminal window:
   ```bash
   cd frontend
   ```
2. Install npm dependencies:
   ```bash
   npm install
   ```
3. Start the Vite React development server:
   ```bash
   npm run dev
   ```
   The portal interface will launch at `http://localhost:5173`.

---

## 🐳 Docker Deployment Guide

For production-like deployment without installing python or node local environments, compile everything using Docker Compose:

1. In the project root, build and launch containers:
   ```bash
   docker-compose up --build
   ```
2. The containers will automatically:
   * Build the backend, download base weights, and listen on port `8000`.
   * Build the frontend, bundle Vite assets, and serve them via Nginx on port `80`.
3. Open `http://localhost` in your browser to access the application.

## 🔍 Forensic Analysis Workflow

1. **Load Dashboard**: Access the operations center to review steganographic context.
2. **Execute Scan**: Open the Detection Engine, choose your classifier (e.g. Vision Transformer), drop in an image (JPG/PNG), and click **Execute Steg-Forensic Scan**.
3. **Review Metrics & Residuals**: Check the prediction confidence score and compare high-pass noise overlays.
4. **Examine Explainability**: Switch to the Explainable AI sub-workspace to analyze heatmaps, coalitions, and regression boundaries.
5. **Export Evidence**: Open the Reports tab, click **Compile Forensic PDF**, and download your finalized evidence log.
